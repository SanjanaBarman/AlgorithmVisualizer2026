Linear Search

Linear Search checks each element one by one. Time Complexity: O(n)

C Code


int linearSearch(int arr[], int n, int key) {
    for (int i = 0; i < n; i++) {
        if (arr[i] == key)
            return i;
    }
    return -1;
}

C++ Code

Java Code


class LinearSearch {
    static int linearSearch(int arr[], int key) {
        for (int i = 0; i < arr.length; i++)
            if (arr[i] == key)
                return i;
        return -1;
    }
}
