Insertion Sort

Insertion Sort builds the final sorted array one item at a time. Time Complexity: O(n²)

C Code


#include <stdio.h>
void insertionSort(int arr[], int n) {
    for(int i = 1; i < n; i++) {
        int key = arr[i];
        int j = i - 1;
        while(j >= 0 && arr[j] > key) {
            arr[j+1] = arr[j];
            j--;
        }
        arr[j+1] = key;
    }
}
